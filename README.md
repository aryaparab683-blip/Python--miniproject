# 📚 Study Tracker System

A **Flask-based web application** designed to help students manage their study routines efficiently. Track tasks, goals, sessions, and boost productivity with built-in tools like a **Pomodoro timer**, analytics charts, and a calendar view.

## 🚀 Project Overview

The **Study Tracker System** helps students:

* Organize study tasks and subjects
* Track progress and productivity
* Manage notes and long-term goals
* Improve focus using the Pomodoro technique



## 🎯 Objectives

* Build a **full-stack web application** using Flask

* Implement secure **authentication system**

## 🗄️ Database Overview

The application uses **SQLite3** with the following entities:

* **Users** – User account & profile data
* **Tasks** – Daily or scheduled tasks
* **Goals** – Long-term objectives
* **Subjects** – Study categorization
* **Sessions** – Study session tracking
* **Notes** – User notes storage


## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone <your-repo-link>
cd study-tracker
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run Application

```bash
python main.py
```

### 4️⃣ Open in Browser

```
http://127.0.0.1:5000
```
